# Frontend - Sistema de Controle Financeiro

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

O frontend estará rodando em `http://localhost:5173`

## Build

```bash
npm run build
```

## Estrutura

```
src/
├── lib/
│   ├── api/          # Cliente API
│   ├── components/   # Componentes reutilizáveis
│   ├── stores/       # Svelte stores
│   └── utils/        # Utilitários
└── routes/           # Páginas (SvelteKit)
```
